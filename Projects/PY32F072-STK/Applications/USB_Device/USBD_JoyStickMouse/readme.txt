================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了USB鼠标功能
USB 鼠标主要是借助 USB HID 类实现，将其模拟成一个 USB 鼠标设备，设备程序每隔1s钟
向主机发送一次鼠标相对位置，在PC端，可以看到鼠标每隔1s移动一次位置。

================================================================================
测试环境：
测试用板：STK
MDK版本： 5.28
================================================================================
使用步骤：
1. 下载并运行程序；
2. 通过usb连接线，一端连接STK板，另一端连接PC电脑；
3. PC端鼠标每隔1s钟移动一次位置；

================================================================================
注意事项：
1. 用户测试程序过程中，如果对USB接口进行热插拔，当重新插上USB线到PC端后，有可能鼠标不
再移动，是因为程序每次发送新坐标位置之前会判断上一次是否发送成功（检查标志位
HID_STATE_BUSY），如果热插拔过程中导致USB通讯失败一次，则会停止更新鼠标位置，程序
中取消对HID_STATE_BUSY标志位的判断，即可实现热插拔不影响鼠标移动。

================================================================================
