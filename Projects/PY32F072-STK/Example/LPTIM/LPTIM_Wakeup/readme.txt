================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了LPTIM连续模式事件唤醒STOP模式。

================================================================================
测试环境：
测试用板：PY32F072_STK
MDK版本： 5.28
IAR版本： 9.20

================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 按下按键进入stop模式；
3. 200ms后唤醒stop模式并循环进入stop模式再唤醒；
4. LED保持2.5Hz频率闪烁；
